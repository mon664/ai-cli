Quick Installation

Get started in seconds with a single command

Install via Cargo

cargo install ai-cli

Build from Source

git clone https://github.com/your-username/ai-cli.git
cd ai-cli
cargo build --release

Prerequisites: Rust 1.91+ and Git 2.0+

How to Use

Simple commands for powerful Git workflow automation

Initialize Configuration 

ai-cli init --model openai --openai-key YOUR_API_KEY

Set up AI CLI with your preferred AI model and API keys.

Generate Commit Messages 

git add .
ai-cli commit

Let AI analyze your changes and generate professional commit messages.

Explain Changes 

ai-cli explain --detailed --format markdown

Get detailed explanations of code changes in various formats.

Check Configuration 

ai-cli config --verbose

View your current AI CLI configuration and settings.
